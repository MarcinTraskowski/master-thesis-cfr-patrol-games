# SHIELD Evaluation

This directory contains the scripts used to evaluate strategies generated by the original [SHIELD](https://github.com/anagorko/stackelberg-games/tree/main/stackelberg-games-patrolling) framework. We were able to successfully reproduce the results reported in the corresponding paper.

The evaluation scripts for the finite- and infinite-horizon settings are located in

```text
analysis/
```

## Generating SHIELD strategies

SHIELD strategies can be generated using the original `scripts/sgp_compute_schedules.py` script with the appropriate parameters.

The original script is configured to generate strategies for the Gdynia environment. To generate strategies for the San Francisco environment, the following helper function must be added:

```python
def solve_and_save_sf():
    out_dir_path = sgp.directories.strategy_dir
    os.makedirs(out_dir_path, exist_ok=True)

    output_name = f"{out_dir_path}san_francisco.pickle"

    if os.path.exists(output_name):
        print(f"{output_name} already exists, skipping.")
        return

    tau = {
        0: 8, 1: 6, 2: 11, 3: 10,
        4: 6, 5: 10, 6: 9, 7: 10,
        8: 11, 9: 9, 10: 10, 11: 8
    }

    environment = sgp.john_et_al(12, self_loops=False)
    data = sgp.patrolling_problem(environment, 1, tau)

    solver = sgp.BisectSolver(10**-3, True)
    solution = solver.solve(data)

    with open(output_name, "wb") as output_file:
        pickle.dump(solution, output_file)

    print(f"Wrote strategy to {output_name}")
```

The parameters of the San Francisco environment, including the attack durations, were taken from the original `scripts/uai_monte_carlo_experiments.py` script.
