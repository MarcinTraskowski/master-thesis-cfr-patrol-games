# Applying Counterfactual Regret Minimization to Patrol Games

This repository contains the source code accompanying my master's thesis:

> **Applying Counterfactual Regret Minimization to Patrol Games**

The goal of this work is to investigate the applicability of Counterfactual Regret Minimization (CFR) algorithms to robotic patrol games. In particular, the thesis studies the defender strategies produced by vanilla CFR, CFR+, and Monte Carlo CFR. The patrol game is implemented on top of the [OpenSpiel](https://github.com/google-deepmind/open_spiel), while [SHIELD](https://github.com/anagorko/stackelberg-games/tree/main/stackelberg-games-patrolling) and [RoSSO](https://github.com/conhugh/RoSSO) are used as benchmark methods for comparison.

---

## Repository structure

The repository consists of three main directories. Each directory contains its own README with detailed instructions and additional information.

### `open_spiel`

A fork of OpenSpiel containing the C++ implementation of the patrol game developed in this thesis, together with the files used to train CFR strategies and the analysis scripts used to process experimental results.

### `rosso`

Scripts used to reproduce and evaluate strategies generated by the RoSSO framework.

### `shield`

Scripts used to process and evaluate strategies produced by the SHIELD framework.
